# Easy Home

Easy Home is a modern web application designed to simplify finding and booking professional home services. Whether you need a plumber, electrician, cleaner, or handyman, Easy Home connects you with verified service providers quickly and efficiently.

## Live Site
[Easy Home Live](https://easyhouse-f443f.web.app/)  

## Key Features
- **Service Discovery:** Browse and search for a wide range of home and outside services.  
- **Booking Management:** Book services easily and track your bookings in one place.  
- **Provider Dashboard:** Service providers can add, update, and manage their services.  
- **User Reviews & Ratings:** Users can leave ratings and reviews for services to help others make informed decisions.  
- **Responsive Design:** Fully responsive and mobile-friendly interface for seamless usage on all devices.  
- **Secure Authentication:** Users and service providers can securely sign up, log in, and manage their accounts.  
- **Real-time Updates:** Get live updates on service availability and booking status.  

## Technologies Used
- Frontend: React, Tailwind CSS, Framer Motion  
- Backend: Node.js, Express  
- Database: MongoDB
